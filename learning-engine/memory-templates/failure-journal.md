# Failure Journal

Each entry is a trap — something that looked right but was wrong. Entries include trigger conditions so /antipattern can match them against current work.

The value of this file is not in reading it. The value is that /antipattern reads it FOR you, at the exact moment you're about to repeat a mistake. Every entry is a future intervention.
